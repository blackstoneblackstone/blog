## usort， create_function
> 该问题主要使用的是 create_function() 方法


```php

class Test{
 public  function s(){
  $res=arry()
  array_push($res,array('id'=>2))
  array_push($res,array('id'=>1))
  
  $cmp_func = create_function('$a,$b', '
	if($a["id"]  ==  $b["id"])  return  0;
	return  $a["id"]>$b["id"]?-1:1;');
        // 对结果排序 保证目录在前面
  usort($res, $cmp_func);
    }
}

```
